## COVID 19 OVERVIEW AND ANALYSIS

This project focuses on the analysis of the global COVID-19 pandemic using real-world datasets to explore infection trends, mortality rates, vaccination progress, and regional impacts over time.

COVID-19 (Coronavirus Disease 2019), caused by the SARS-CoV-2 virus, has significantly affected public health systems and economies worldwide. Through data-driven analysis, we can better understand how the virus spread across countries, identify high-risk regions, and evaluate the effectiveness of healthcare and vaccination responses.
